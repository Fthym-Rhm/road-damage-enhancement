# Road Damage Detection – Frame Enhancement Pipeline
This project implements an image enhancement pipeline for improving road surface frames extracted from video recordings. The pipeline includes grayscale conversion, mean filtering, log transformation, and contrast stretching to enhance crack visibility. The enhanced frames are evaluated using image quality metrics such as MSE, PSNR, and SSIM.

## Enhancement Pipeline
Video Input  
→ Frame Extraction  
→ Grayscale Conversion  
→ Mean Filtering  
→ Log Transformation  
→ Contrast Stretching  
→ Evaluation Metrics (MSE, PSNR, SSIM)

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib
